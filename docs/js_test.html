jQueryをRmd上で使ってみる

以下のコードを準備します:

{js} $(function(){ $("#hoge").click(function(){ $(this).fadeOut(1000) }) })

$(function(){
  $("#hoge").click(function(){
    $('.r').fadeOut(1000)
  })
})

そして,以下のコードを利用してボタンを準備します:

<button type="button" class="btn btn-primary" id="hoge">Primary</button>

このボタンを押すと,このページで表示していたRのコードブロックがフェードアウトします。
